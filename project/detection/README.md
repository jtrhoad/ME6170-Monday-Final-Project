# ME 6170 - Detection Program


We are in charge of setup and integration of all sensor data.

## Tasks List

0. Get access to RasberryPi 5
    - Set up serial connection
    - Set up live telemetry feed (wireless data feed)
    - Set up SSH between laptop and RP
1. Read data
    - Ultrasonic Sensor
    - Color Sensor
    - IR Sensor
2. Set up live video feed from camera
3. Detect obstacles
    - Camera object detection
    - Ultrasonic distance detection
    - Detect specific colors via camera
4. Use sensor data to change pathing
5. Use camera to detect specific gesture to indicate action or endpoint to move to.
