# ME6170 — Monday Final Project

A Raspbot V2 mobile ground robot has been procured for each laboratory team to serve as the basis for the design project. This kit contains a metal chassis with Mecanum wheels, 2 DOF PTZ camera, ultrasonic and infrared sensors, buzzer, RGB light bar, and Raspberry Pi-based hardware/software. The AI vision system enables color detection, target and object tracking, and gesture recognition. Open-source tutorials cover many operations. Teams have many possible directions to proceed with this specific project including robot operation, path following, programming, additive manufacturing supplemental components, and/or class competition to complete a task.

## Project Deliverables and Weighting

1. The robot should be able to move and transport an item from one location to another. (45%)
2. The robot should be able to detect and avoid obstacles between the pickup and delivery locations. (35%)
3. The robot should be able to identify when to begin operation based on human gestures. (15%)
4. The robot should be able to pick up and deposit the requested item. (5%)

## Getting Started

- In the RaspberryPi terminal, type "goproject." This will open and set up the virtual enviornment to run programs.
- Any libraries should be downloaded within the venv and not globally.
- Save any python files to their respective project files (detection, interaction, and/or movement)
